# Literature Review Skill

Panduan sistematis melakukan Systematic Literature Review (SLR).

## Workflow SLR

### Fase 1: Planning
1. **Definisi Research Questions**
   - Gunakan framework PICO (Population, Intervention, Comparison, Outcome)
   - Atau SPIDER (Sample, Phenomenon of Interest, Design, Evaluation, Research type)

2. **Protokol Review**
   - Tentukan kriteria inklusi/eksklusi
   - Pilih database yang akan digunakan
   - Tentukan timeframe publikasi

### Fase 2: Searching
1. **Database Akademik**
   - Scopus
   - Web of Science
   - PubMed
   - Google Scholar
   - IEEE Xplore
   - DOAJ

2. **Search Strategy**
   ```
   ("keyword 1" OR "synonym 1") 
   AND 
   ("keyword 2" OR "synonym 2")
   AND
   ("keyword 3")
   ```

3. **Boolean Operators**
   - AND: mempersempit hasil
   - OR: memperluas hasil
   - NOT: mengecualikan

### Fase 3: Screening
1. **Title & Abstract Screening**
   - Baca judul dan abstrak
   - Terapkan kriteria inklusi/eksklusi
   - Catat alasan eksklusi

2. **Full-text Assessment**
   - Baca artikel lengkap
   - Evaluasi kualitas metodologi
   - Keputusan final inklusi

### Fase 4: Data Extraction
1. **Extraction Form**
   | Field | Deskripsi |
   |-------|-----------|
   | Author(s) | Nama penulis |
   | Year | Tahun publikasi |
   | Title | Judul artikel |
   | Journal | Nama jurnal |
   | Objective | Tujuan penelitian |
   | Method | Metodologi |
   | Sample | Ukuran sampel |
   | Key Findings | Temuan utama |
   | Limitations | Keterbatasan |

2. **Quality Assessment**
   - CASP (Critical Appraisal Skills Programme)
   - JBI (Joanna Briggs Institute)
   - Newcastle-Ottawa Scale

### Fase 5: Synthesis
1. **Narrative Synthesis**
   - Kelompokkan berdasarkan tema
   - Identifikasi pola dan tren
   - Bandingkan temuan antar studi

2. **Meta-Analysis** (jika applicable)
   - Hitung effect size
   - Forest plot
   - Heterogeneity test
   - Funnel plot (publication bias)

## Tips

- Simpan semua search history
- Gunakan reference manager (Mendeley/Zotero)
- Dokumentasi setiap keputusan
- Libatkan 2 reviewer untuk reliability
- Update search sebelum submission
