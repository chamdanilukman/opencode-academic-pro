# Thesis Writing Skill

Panduan sistematis menulis tesis dan disertasi berkualitas.

## Workflow Penulisan Tesis

### Fase 1: Persiapan (Minggu 1-2)
1. Tentukan topik dan scope penelitian
2. Lakukan preliminary literature review
3. Identifikasi research gap
4. Rumuskan research questions/hypotheses
5. Diskusi dengan pembimbing

### Fase 2: Proposal (Minggu 3-6)
1. Tulis BAB I - Pendahuluan
2. Tulis BAB II - Tinjauan Pustaka
3. Tulis BAB III - Metodologi
4. Siapkan instrumen penelitian
5. Seminar proposal

### Fase 3: Pengumpulan Data (Minggu 7-12)
1. Uji validitas dan reliabilitas instrumen
2. Kumpulkan data sesuai metodologi
3. Dokumentasi proses pengumpulan data
4. Backup data secara berkala

### Fase 4: Analisis & Penulisan (Minggu 13-18)
1. Analisis data
2. Tulis BAB IV - Hasil dan Pembahasan
3. Tulis BAB V - Penutup
4. Revisi BAB I-III jika diperlukan

### Fase 5: Finalisasi (Minggu 19-20)
1. Review keseluruhan
2. Proofreading dan editing
3. Format sesuai panduan universitas
4. Persiapan sidang

## Template BAB

### BAB I - Pendahuluan
```markdown
1.1 Latar Belakang Masalah
    - Konteks umum (global  nasional  lokal)
    - Fenomena/masalah yang diteliti
    - Data pendukung (statistik, fakta)
    - Urgensi penelitian

1.2 Identifikasi Masalah
    - Daftar masalah yang teridentifikasi

1.3 Batasan Masalah
    - Fokus penelitian
    - Variabel yang diteliti
    - Lokasi dan waktu

1.4 Rumusan Masalah
    - Pertanyaan penelitian (research questions)

1.5 Tujuan Penelitian
    - Tujuan umum
    - Tujuan khusus

1.6 Manfaat Penelitian
    - Manfaat teoritis
    - Manfaat praktis
```

### BAB II - Tinjauan Pustaka
```markdown
2.1 Landasan Teori
    - Teori utama
    - Teori pendukung
    - Definisi konseptual

2.2 Penelitian Terdahulu
    - Minimal 10 penelitian relevan
    - Tabel perbandingan penelitian

2.3 Kerangka Pemikiran
    - Alur logika penelitian
    - Diagram kerangka pemikiran

2.4 Hipotesis (jika kuantitatif)
    - H1, H2, H3, dst.
```

### BAB III - Metodologi
```markdown
3.1 Desain Penelitian
    - Pendekatan (kuantitatif/kualitatif/mixed)
    - Jenis penelitian

3.2 Populasi dan Sampel
    - Definisi populasi
    - Teknik sampling
    - Ukuran sampel

3.3 Variabel Penelitian
    - Variabel independen
    - Variabel dependen
    - Variabel kontrol (jika ada)

3.4 Definisi Operasional
    - Tabel operasionalisasi variabel

3.5 Instrumen Penelitian
    - Jenis instrumen
    - Kisi-kisi instrumen
    - Uji validitas dan reliabilitas

3.6 Teknik Pengumpulan Data
    - Prosedur pengumpulan
    - Sumber data

3.7 Teknik Analisis Data
    - Analisis deskriptif
    - Analisis inferensial
    - Software yang digunakan
```

## Checklist Kualitas

- [ ] Research questions jelas dan terukur
- [ ] Literature review komprehensif dan up-to-date
- [ ] Metodologi sesuai dengan research questions
- [ ] Analisis data menjawab semua research questions
- [ ] Kesimpulan berdasarkan temuan penelitian
- [ ] Sitasi konsisten dan lengkap
- [ ] Format sesuai panduan universitas
